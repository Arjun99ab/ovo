# OvO Modloader

A feature rich, gui integrated OvO mod menu.

## Features
- Enable / disable mods
- Add custom mods
- Favorite mods
- Customize mod settings (keybinds, etc.)
- Saves data & mods cross sessions & versions
- Supports 9+ versions
