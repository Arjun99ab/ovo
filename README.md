# OvO Modloader

A feature rich, gui integrated OvO mod menu.

![modloader screen](./src//img/github/screen.png)

## Features
- Enable / disable mods
- Add custom mods
- Favorite mods
- Customize mod settings
- Use community made skins
- Saves data & mods cross sessions & versions
- Supports 9+ versions
